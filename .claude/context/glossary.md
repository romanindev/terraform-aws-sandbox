# Glossary

## Todo

A simple task item used as the project domain model.

## API

The NestJS backend application located in `apps/api`.

## Web

The React + Vite frontend application located in `apps/web`.

## Infrastructure

AWS resources managed with Terraform under `infra/terraform`.

## Day

A focused implementation step from ROADMAP.md. Each day should be handled as a small, logical unit of work.

## AI Skill

A reusable instruction file under `.claude/skills` that guides an AI assistant for a specific type of task.
